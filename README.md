# CLI Video Editor Documentation

## Description of the project

This project provides a command-line interface (CLI) video editing tool that enables users to perform common video editing operations programmatically or interactively. Built on top of the MoviePy library, the tool supports operations such as zooming, muting audio segments, spatial cropping (frame regions), and temporal cropping (time segments).

The core `VideoEditor` class manages a sequence of editing operations, applying them in a specific order to ensure consistent results. The CLI interface allows users to interactively select and configure these operations, making it accessible for both technical and non-technical users.
