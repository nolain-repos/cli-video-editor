# Video Editor Documentation

## Description of the project

The Video Editor project is a Python-based tool designed for automated video editing and manipulation. It leverages the MoviePy library to provide a programmatic interface for applying various video editing operations, including zooming, cropping (both temporal and spatial), and audio muting.

The project features an interactive terminal-based interface that allows users to select and configure editing actions through keyboard navigation. The core `VideoEditor` class manages the sequence of operations and applies them in a specific order to ensure consistent results. This solution is ideal for automating repetitive video editing tasks, processing multiple video files, or integrating video editing capabilities into larger workflows.
